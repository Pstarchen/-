# 标签学习

## html标签结构

```
<strong>文字加粗</strong>	双标签
开始标签 包裹的内容 结束标签
```

少数标签由一部分组成，叫做单标签

例：

```
<br>换行
<hr>创建一条水平线
```

规律：若需要确定开始位置和结束位置，则是双标签，反之则为单标签

## 标签的关系

父子关系

```html
<head>
    <title></title>
</head>
```

兄弟关系

```html
<head></head>
<body></body>
```

## 标题标签

代码：h系列标签

```html
<h1>
    1级标题
</h1>
...
<h6>
    2级标题
</h6>
```

语义：1~6级标题	重要程度依次递减

特点：独占一行

在body中输入`h1`回车，即可生成<h1></h1>

ctrl+D可选中同一标签内所有选中元素，可同时操作
